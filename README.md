## Color Theory
### Moods
- Red     : 🔥   - Love, Energy, Intensity
- Yellow  : 🌞  - Joy, Intellect, Attention
- Green   : 🍀  - Freshness, Safety, Growth
- Blue    : 💧   - Stability, Trust, Serenity
- Purple  : 👑  - Royalty, Wealth, Femininity
